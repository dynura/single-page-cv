# Single-Page CV

This repository contains my completed solution to the [Single-Page CV Project](https://roadmap.sh/projects/single-page-cv) challenge on roadmap.sh.

## Project Overview
The objective of this project is to build a structured, semantic, single-page professional curriculum vitae (CV) using **pure HTML5**. This project serves as a foundational step to master document outlines, SEO metadata, and structural nesting prior to adding styling layers.

## Features & Requirements Met
- **Semantic HTML5 Layout:** Structured using machine-readable components including `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` instead of generic container blocks.
- **SEO & Metadata:** Implemented vital standard SEO tags (`description`, `keywords`, `author`) inside the head block to maximize search engine indexing efficiency.
- **Open Graph Protocol:** Configured Open Graph (`og:title`, `og:description`, `og:type`) tags to format clean preview cards when shared on social media platforms.
- **Custom Favicon:** Linked a modular browser icon snippet to establish immediate branding identifiers within user tab panes.

## Setup & Preview
To preview this project locally, simply clone the repository and open `index.html` in any modern web browser.
